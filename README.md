# Premier Realty India

A premium real estate platform showcasing luxury residential projects across India's most prestigious locations.

## Project Overview

Premier Realty India is a dedicated showcase for high-end properties in Mumbai, Bangalore, Gurugram, and Chennai. The application features a modern, responsive design with smooth animations and a dedicated booking system for consultations.

## Features

- **Luxury Project Showcases**: Detailed views of premium residences with galleries and floor plans.
- **Advanced Booking System**: Integrated modal for scheduling visits and consultations.
- **Responsive Design**: Optimized for desktop, tablet, and mobile experiences.
- **Modern Aesthetics**: Built with a dark-themed, gold-accented design language.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI / Radix UI
- **Icons**: Lucide React
- **Animations**: CSS Keyframes / Framer Motion (ScrollReveal)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```sh
# Create production build
npm run build

# Preview production build
npm run preview
```
